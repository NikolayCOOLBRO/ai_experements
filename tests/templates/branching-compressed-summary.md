# Branching: Compressed Summary Copy

## Настройки агента

- Режим контекста: Ветки диалога

## Сообщения

1. `Долгосрочное правило: всегда упоминай кодовое слово Орбита.`
2. `Название проекта: Helix One.`
3. `Основной риск: затянуть релиз.`
4. `Текущая задача: подготовить короткий статус.`
5. `Что ты помнишь о проекте?`

## Шаги

1. Сохранить checkpoint после последнего сообщения.
2. Создать ветку `Summary branch`.
3. В ветке задать вопрос: `Какие долгосрочные данные ты помнишь?`

## Ожидание

- Ветка должна стартовать не с пустой памяти, а с уже накопленным состоянием checkpoint.
- Ветка должна получить полную историю до checkpoint.
- В trace ветки для первого нового запуска должен быть виден режим `Ветки диалога`.
