# Branching: Isolation Between Branches

## Настройки агента

- Режим контекста: Ветки диалога

## Подготовка

1. Создать чат.
2. Отправить сообщение: `Запомни базовые условия: продукт = Nova Desk, рынок = B2B. Ответь: записал.`
3. Сохранить checkpoint.
4. Создать две ветки: `Ветка A` и `Ветка B`.

## Сообщения в ветке A

1. `В этой ветке новая стратегия: идем в enterprise-сегмент. Ответь: стратегия A записана.`
2. `Что ты помнишь о стратегии этой ветки?`

## Сообщения в ветке B

1. `В этой ветке новая стратегия: идем в SMB-сегмент. Ответь: стратегия B записана.`
2. `Что ты помнишь о стратегии этой ветки?`

## Ожидание

- Ветка A отвечает только про enterprise-сегмент.
- Ветка B отвечает только про SMB-сегмент.
- Сообщения из ветки A не попадают в историю ветки B.
- Сообщения из ветки B не попадают в историю ветки A.
