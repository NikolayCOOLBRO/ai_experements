# Sliding Window: базовая проверка

## Настройки агента

- Режим контекста: Скользящее окно
- Окно памяти: 3
- Summary: любое значение, не должно использоваться

## Сообщения

1. `Факт A: мой кодовый цвет - янтарный. Ответь только: принято A.`
2. `Факт B: мой кодовый город - Казань. Ответь только: принято B.`
3. `Факт C: мой кодовый фрукт - груша. Ответь только: принято C.`
4. `Факт D: мой кодовый инструмент - компас. Ответь только: принято D.`
5. `Назови все кодовые факты, которые доступны тебе из текущего контекста.`

## Ожидание

В prompt последнего запуска должны попасть только сообщения 3, 4 и 5. Агент не должен уверенно называть факты A и B.
