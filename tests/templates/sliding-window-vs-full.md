# Sliding Window vs Full

## Настройки агента A

- Режим контекста: Без сжатия
- Окно памяти: 3

## Настройки агента B

- Режим контекста: Скользящее окно
- Окно памяти: 3

## Сообщения для обоих агентов

1. `Запомни контрольное слово: север. Ответь: записал.`
2. `Запомни контрольное число: 481. Ответь: записал.`
3. `Запомни контрольный предмет: фонарь. Ответь: записал.`
4. `Запомни контрольное животное: выдра. Ответь: записал.`
5. `Какие контрольные данные ты видишь сейчас? Если данных нет в контексте, не угадывай.`

## Ожидание

Агент A в режиме full может использовать всю историю. Агент B в режиме sliding-window должен видеть только последние 3 сообщения и не должен уверенно вспоминать первые два факта.
